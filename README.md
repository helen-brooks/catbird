Catbird
-------

A code for generation of Python objects and input for various [MOOSE](https://moose.inl.gov/SitePages/Home.aspx).

Prerequisites
-------------

  - Python version >=3.6
  - [NumPy](https://numpy.org/)
  - `pytest` (for testing)

Installation
------------

From a terminal, run

```bash
pip install .
```

To include packages for testsing, run

```bash
pip install .[test]
```
